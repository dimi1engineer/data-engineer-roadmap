Fase 3: entrega profissional

    Prática: transformar um projeto local em um fluxo com testes e deploy controlado.

Entrega da fase: repositório com estrutura profissional, testes e pipeline de entrega.
Fase 4: cloud e infraestrutura

    IaC: noções de Terraform ou CloudFormation para automatizar infraestrutura.

    Prática: subir uma solução simples na AWS com configuração repetível.

Fase 5: MLOps e governança
